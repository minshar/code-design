עיצוב קוד: זכרון דברים

בעמוד זה יהיה סיכום של כל הדברים שנלמד בקורס. השתמשו במידע הזה כדי להיזכר בתגים ובפקודות השונות. מומלץ לשמור על העמוד פתוח בטאב כלשהו ברקע בזמן העבודה, כדי שתוכלו לחזור ולבדוק דברים לפי הצורך. קידוד נעים!

מונחים כלליים

המונח ההסבר
WWW (World Wide Web) ה״ווב״ - הפרוטוקול האינטרנטי לצפייה באתרי אינטרנט
HTML - HyperText Markup Language שפת תיוג היפרטקסט - הדרך שבה מגדירים תוכן באתרי אינטרנט
CSS - Cascading Style Sheets גיליונות עיצוב משורשרים - הדרך שבה מגדירים עיצוב באתרי אינטרנט
HTTP - HyperText Transfer Protocol שם הפרוטוקול עצמו של ה-WWW
URL - Uniform Resource Locator כתובת של אתר ברשת האינטרנט
JavaScript שפת תכנות להגדרת התנהגות באתרי אינטרנט
Script כך נקראים קבצי הקוד בג׳אווהסקריפט (או בשפות אחרות)
Server / Host שרת - מחשב המנגיש אתר דרך רשת האינטרנט
Browser/ Client דפדפן - תוכנה לצפייה באתרי אינטרנט (כמו גוגל כרום, ספארי, אדג׳)
Viewport כלי לבדיקה ובחינת אתרי אינטרנט בחלון הדפדפן עצמו
Inspector עץ האובייקטים - הדרך שבה הדפדפן מפענח את ה-HTML
DOM Tree תגית HTML - כל הוראה המופיעה ברצף ה-HTML בין סימני ><
Attribute תכונה - פרמטר שמוגדר בתגית HTML, למשל href בתגית קישור (a)
Element אלמנט - כל מה שמוכל בתגית HTML, כולל התגית עצמה והתכונות שלה; אלמנטים יכולים להיות מוכלים בתוך אלמנטים אחרים (כחלק מעץ האובייקטים)
Front End התוכן שמופיע באתר לקהל שנכנס אליו
Back End התוכן הנגיש למנהלי האתר בלבד - למשל, ממשק הניהול
