בעמוד זה יהיה סיכום של כל הדברים שנלמד בקורס. השתמשו במידע הזה כדי להיזכר בתגים ובפקודות השונות. מומלץ לשמור על העמוד פתוח בטאב כלשהו ברקע בזמן העבודה, כדי שתוכלו לחזור ולבדוק דברים לפי הצורך. קידוד נעים!

מונחים כלליים

המונח ההסבר
WWW (World Wide Web) ה״ווב״ - הפרוטוקול האינטרנטי לצפייה באתרי אינטרנט
HTML - HyperText Markup Language שפת תיוג היפרטקסט - הדרך שבה מגדירים תוכן באתרי אינטרנט
CSS - Cascading Style Sheets גיליונות עיצוב משורשרים - הדרך שבה מגדירים עיצוב באתרי אינטרנט
HTTP - HyperText Transfer Protocol שם הפרוטוקול עצמו של ה-WWW
URL - Uniform Resource Locator כתובת של אתר ברשת האינטרנט
JavaScript שפת תכנות להגדרת התנהגות באתרי אינטרנט
Script כך נקראים קבצי הקוד בג׳אווהסקריפט (או בשפות אחרות)
Server / Host שרת - מחשב המנגיש אתר דרך רשת האינטרנט
Browser/ Client דפדפן - תוכנה לצפייה באתרי אינטרנט (כמו גוגל כרום, ספארי, אדג׳)
Viewport החלק בחלון או טאב הדפדפן המציג את האתר (או של תצוגת המובייל)
Inspector כלי לבדיקה ובחינת אתרי אינטרנט בחלון הדפדפן עצמו
DOM Tree עץ האובייקטים - הדרך שבה הדפדפן מפענח את ה-HTML
Attribute תכונה - פרמטר שמוגדר בתגית HTML, למשל href בתגית קישור (a)
HTML Tag תגית HTML - כל הוראה המופיעה ברצף ה-HTML בין סימני ><
Element אלמנט - כל מה שמוכל בתגית HTML, כולל התגית עצמה והתכונות שלה; אלמנטים יכולים להיות מוכלים בתוך אלמנטים אחרים (כחלק מעץ האובייקטים)
Front End התוכן שמופיע באתר לקהל שנכנס אליו
Back End התוכן הנגיש למנהלי האתר בלבד - למשל, ממשק הניהול

כתיבת קורות חיים / ויסלבה שימבורסקה

מַה נָּחוּץ?
נָחוּץ לִכְתֹּב בַּקָּשָׁה
וּלְצָרֵף לָהּ קוֹרוֹת חַיִּים.

לְלֹא קֶשֶׁר לְאֹרֶךְ הַחַיִּים
עַל הַקּוֹרוֹת לִהְיוֹת קְצָרוֹת.

תִּמְצוּת הָעֻבְדּוֹת וּבְחִירָתָן הֶכְרֵחִיִּים.
הֲמָרַת נוֹפִים בִּכְתוֹבוֹת
וְזִכְרוֹנוֹת רוֹפְפִים בְּתַאֲרִיכִים מְקֻבָּעִים.
מִכָּל הָאֲהָבוֹת לִרְשֹׁם אֶת הַנִּשּׂוּאִין בִּלְבַד,
וּמֵהַיְּלָדִים רַק אֶת אֵלֶּה שֶׁנּוֹלְדוּ.

זֶה שֶׁמַּכִּיר אוֹתְךָ חָשׁוּב יוֹתֵר מִזֶּה שֶׁאַתָּה מַכִּיר.
טִיּוּלִים רַק בְּמִדָּה וּלְחוּץ לָאָרֶץ.
הִשְׁתַּיְּכוּת לְמָה אַךְ בְּלֹא הַלָּמָה.
צִיּוּנִים לְשֶׁבַח בְּלֹא הַנִּמּוּקִים.

כְּתֹב כְּאִלּוּ מֵעוֹלָם לֹא דִּבַּרְתָּ עִם עַצְמְךָ
וּכְאִלּוּ עָקַפְתָּ עַצְמְךָ מֵרָחוֹק.

הִתְעַלֵּם מִכְּלָבִים, חֲתוּלִים וְצִפֳּרִים,
מַזְכָּרוֹת נוֹשָׁנוֹת, חֲבֵרִים וַחֲלוֹמוֹת.

מְחִיר וְלֹא עֶרֶךְ
כּוֹתֶרֶת וְלֹא תֹּכֶן.
מִסְפַּר הַנַּעֲלַיִם וְלֹא הַיַּעַד שֶׁאֵלָיו הוֹלֵךְ
זֶה שֶׁאָמוּר לִהְיוֹת אַתָּה.

לְכָךְ יֵשׁ לְצָרֵף תַּצְלוּם בְּאֹזֶן גְּלוּיָה.
רַק צוּרָתָהּ נִלְקַחַת בְּחֶשְׁבּוֹן, לֹא מַה שֶּׁנִּשְׁמָע.
מַה נִּשְׁמָע?
נַהֲמַת מְכוֹנוֹת הַגּוֹרְסוֹת אֶת הַנְּיָר.

כיוון הטקסט ltr / rtl direction: rtl;
שם הפונט "font name", serif / sans-serif / monospace font-family: "Arial", sans-serif;
גודל הפונט px / pt / em / % font-size: 20px;
גובה שורה (לדינג) px / pt /em / % line-height: 30px;
משקל הפונט light / normal / bold / 100–900 font-weight: bold;
סגנון הפונט normal / italic font-style: italic;
ריווח בין אותיות px / pt / em / % letter-spacing: 1px;

CSS - יחידות מידה

הסבר יחידת המידה
אפשר תמיד לציין 0 ללא יחידת מידה 0
פיקסלים - במסכי רטינה זו מכפלת פיקסלים בהתאם לרזולוציה האמיתית px
פוינט - דומה לפיקסלים, מתאים יותר לפרינט pt
1 ״אם״ = גודל הטקסט היחסי של האלמנט המכיל. למשל אם באלמנט המכיל גודל הטקסט הוא 16px, אז 1em = 16px em
כמו em רק תמיד ביחס למסמך (ל-body) rem
אחוזים, עבור טקסט זה כמו em ועבור אלמנטים זה בד״כ מרוחב העמוד או האלמנט המכיל %
1/100 מרוחב ה-viewport vw
1/100 מגובה ה-viewport vh
ללא יחידת מידה - זה משתנה, עדיף תמיד לציין את יחידת המידה!

CSS - צבעים

הסבר הגדרת הצבע
קוד hex של האלמנט (אפשר להעתיק מפוטושופ) - לא לשכוח את ה-# #ffffff
קוד rgb סטנדרטי rgb(255, 255, 255)
קוד rgb בתוספת שקיפות - ערוץ alpha מ-0 (שקוף) עד 1 (אטום) rgba(255, 255, 255, 1)
קוד hsl - גוון (0-360), רוויה (0-100%), ובהירות (0-100%) hsl(120, 100%, 100%)
קוד hsl בתוספת שקיפות hsla(120, 100%, 100%, 1)
שמות סטנדרטיים של צבעים (רשימה) white
שקוף (בלתי נראה) transparent
