Front End התוכן שמופיע באתר לקהל שנכנס אליו

מַה נָּחוּץ?
נָחוּץ לִכְתֹּב בַּקָּשָׁה
וּלְצָרֵף לָהּ קוֹרוֹת חַיִּים.

לְלֹא קֶשֶׁר לְאֹרֶךְ הַחַיִּים
עַל הַקּוֹרוֹת לִהְיוֹת קְצָרוֹת.

תִּמְצוּת הָעֻבְדּוֹת וּבְחִירָתָן הֶכְרֵחִיִּים.
הֲמָרַת נוֹפִים בִּכְתוֹבוֹת
וְזִכְרוֹנוֹת רוֹפְפִים בְּתַאֲרִיכִים מְקֻבָּעִים.
מִכָּל הָאֲהָבוֹת לִרְשֹׁם אֶת הַנִּשּׂוּאִין בִּלְבַד,
וּמֵהַיְּלָדִים רַק אֶת אֵלֶּה שֶׁנּוֹלְדוּ.

זֶה שֶׁמַּכִּיר אוֹתְךָ חָשׁוּב יוֹתֵר מִזֶּה שֶׁאַתָּה מַכִּיר.
טִיּוּלִים רַק בְּמִדָּה וּלְחוּץ לָאָרֶץ.
הִשְׁתַּיְּכוּת לְמָה אַךְ בְּלֹא הַלָּמָה.
צִיּוּנִים לְשֶׁבַח בְּלֹא הַנִּמּוּקִים.

כְּתֹב כְּאִלּוּ מֵעוֹלָם לֹא דִּבַּרְתָּ עִם עַצְמְךָ
וּכְאִלּוּ עָקַפְתָּ עַצְמְךָ מֵרָחוֹק.

הִתְעַלֵּם מִכְּלָבִים, חֲתוּלִים וְצִפֳּרִים,
מַזְכָּרוֹת נוֹשָׁנוֹת, חֲבֵרִים וַחֲלוֹמוֹת.

מְחִיר וְלֹא עֶרֶךְ
כּוֹתֶרֶת וְלֹא תֹּכֶן.
מִסְפַּר הַנַּעֲלַיִם וְלֹא הַיַּעַד שֶׁאֵלָיו הוֹלֵךְ
זֶה שֶׁאָמוּר לִהְיוֹת אַתָּה.

לְכָךְ יֵשׁ לְצָרֵף תַּצְלוּם בְּאֹזֶן גְּלוּיָה.
רַק צוּרָתָהּ נִלְקַחַת בְּחֶשְׁבּוֹן, לֹא מַה שֶּׁנִּשְׁמָע.

CSS - יחידות מידה

הסבר יחידת המידה
אפשר תמיד לציין 0 ללא יחידת מידה 0
פיקסלים - במסכי רטינה זו מכפלת פיקסלים בהתאם לרזולוציה האמיתית px
פוינט - דומה לפיקסלים, מתאים יותר לפרינט pt
1 ״אם״ = גודל הטקסט היחסי של האלמנט המכיל. למשל אם באלמנט המכיל גודל הטקסט הוא 16px, אז 1em = 16px em
כמו em רק תמיד ביחס למסמך (ל-body) rem
אחוזים, עבור טקסט זה כמו em ועבור אלמנטים זה בד״כ מרוחב העמוד או האלמנט המכיל %
1/100 מרוחב ה-viewport vw
1/100 מגובה ה-viewport vh
ללא יחידת מידה - זה משתנה, עדיף תמיד לציין את יחידת המידה!

Flexbox

תכונות שנותנים לאלמנט ה״הורה״:

הסבר אפשרויות קוד
מפעיל את מודל ה-flexbox על כל ה״ילדים״ הישירים של האלמנט העוטף display: flex;
יישור אופקי - לקצה התחילי (ימין בעברית, שמאל באנגלית), לקצה הסופי, למרכז, או ריווח שווה בין האלמנטים באופנים שונים flex-start / flex-end / center / space-between / space-around / space-evenly justify-content: center;
יישור אנכי של האייטמים - לקצה העליון, התחתון, למרכז, למתוח את כל האלמנטים או לפי קו הבסיס של הטקסטים flex-start / flex-end / center / stretch / baseline align-items: center;
יישור אנכי של הקבוצה - מה לעשות עם החלל שנותר (אם נותר)? ליישר למעלה, למטה, למרכז או לרווח באופן שווה flex-start / flex-end / center / stretch / space-between / space-around align-content: center;
האם לאפשר שורות מרובות (wrap) או לסדר את כל האלמנטים בשורה אחת (nowrap)? wrap / nowrap / wrap-reverse flex-wrap: wrap;
להפוך את הכיוון - לסדר בטור במקום בשורה (או בטורים/שורות מהסוף להתחלה)? row / row-reverse / column / column-reverse flex-direction: column;

תכונות שנותנים ל״ילדים״:

הסבר אפשרויות קוד
כמה להגדיל/להקטין/מה הגודל ההתחלתי של האלמנט? זה בעצם פירוק של שלושת התכונות הבאות: none / [number] [number] [size] flex: 1;
בכמה להרחיב את האלמנט (ביחס לאחרים) [number] flex-grow: 2;
בכמה להצר את האלמנט (ביחס לאחרים) [number] flex-shrink: 3;
מה הגודל ההתחלתי של האלמנט לפני השינויים px / em / % / auto flex-basis: 200px;
באיזה סדר להציג את האלמנט (ביחס לאחרים) [number] order: 1;
לאן ליישר את האלמנט (אם רוצים ליישר אותו באופן שונה משאר האלמנטים בשורה) auto / flex-start / flex-end / center / baseline / stretch align-self: center;

CSS - מיקום אבסולוטי של אלמנטים

הסבר אפשרויות קוד
מיקום סטטי (רגיל) של אלמנט - בררת המחדל position: static;
מיקום אבסולוטי של אלמנט - ביחס לחלון או לאלמנט הרלטיבי העוטף position: absolute;
